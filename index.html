Umbrella provides a weather information from the Weather Underground service, which as a data source uses official meteorological broadcast services, raw data from airports weather stations, raw data from radars, and raw data from official weather stations. It also uses observations from members with automated personal weather stations. With this application you can view current weather data for any city as well as daily forecast (for 10 day) and hourly forecast (for 36 hours).

Umbrella has the following features:

  • current weather details
  • daily forecast for 10 days
  • hourly forecast for 36 hours
  • set the weather update interval
  • severe weather alerts (US and Europe only) with hub notification
  • list of the nearest weather stations
  • list of the favorites cities and weather stations
  • details of the current weather station
  • weather map of clouds, precipitation, pressure, snow, temperature and wind speed
  • possibility to share a link for the weather for the current location
  • set custom background and foreground

FAQ:

  • An Api Key is a code passed in by application calling an Weather Underground API to identify the calling application to the Weather Underground Web site.
  • The UV is an international standard measurement of the strength of sunburn-producing ultraviolet radiation at a particular place and time.
  • The Feels Like is the general term for the perceived outdoor temperature, caused by the combined effects of air temperature, relative humidity and wind speed.
  • The Actual Pressure is the atmospheric pressure at a given elevation on land. If Show actual pressure option is disabled, the application shows mean sea level pressure.
  • The Weather Map displays various meteorological features across a particular area at a particular point in time and has various symbols which all have specific meanings.
